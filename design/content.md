# GREY VECTOR · 内容设计文档

> AshGrid 3D · 未来战场 · 体素 / 构成主义 / 黑灰白 + 红
>
> 设计原则:抽象化、低多边形、机器人化战争、城市废墟 + 无人化战术网络

---

## 一、玩家 / 机器人单位状态系统

| 英文 / EN | 中文 / CN | 用途 (1 句) | UI 显示建议 |
|---|---|---|---|
| **HP** | 生命值 | 单位结构完整度,归零即失能 | 左下方主血条 + 数字 (HP < 30 转红色脉动) |
| **Armor** | 装甲值 | 优先承受伤害的护甲层,可被破甲弹穿透 | HP 上方薄蓝色子条,装甲穿透时碎块掉落 |
| **Battery** | 电池量 | 移动 / 冲刺 / 控制无人机 / 电子战的能源 | 右下方闪电图示 + 百分比 (< 30 % 橙,< 10 % 红闪) |
| **Signal** | 讯号强度 | 与无人机网路 / 指挥节点的连线品质 | HUD 顶部 4 格天线图示,断线红色 X |
| **Heat** | 热量 / 过热 | 连续射击 / 加速 / EW 操作累积,满载强制冷却 | 武器栏旁温度计图示,白→黄→红渐变 |
| **Stability** | 稳定度 | 命中 / 爆炸 / EMP 后下降,影响瞄准与命中率 | 准星抖动幅度 + 角落小图示百分比 |
| **Mobility** | 机动性 | 速度倍率 (0.5×–1.3×),腿部受损会降低 | 脚印图示 + 倍率,小于 1.0× 时黄色 |
| **Sensor** | 感测能力 | 雷达 / 热成像范围 / 敌人辨识清晰度 | 雷达脉冲图示 + 公尺数,影响小地图覆盖范围 |
| **Payload** | 载重 | 装备总重,超载会扣 Mobility 与 Battery | 货箱图示 + 进度条 (kg),超载红色警告 |
| **Noise** | 噪音值 | 被敌方被动侦测的概率,冲刺 / 开火会上升 | 声波图示 + 等级,数值高时音效 / 屏幕脉动 |
| **Suppression** | 压制状态 | 持续被敌火压制时的临时降级状态 | 屏幕暗角 + 红色压制图示 + 命中率下降提示 |
| **Jammed** | 被干扰状态 | 进入敌方 EW 范围,无法使用无人机 / 小地图 | 全屏红色边框 + 闪电-X 图示 + 雷达雪花 |
| **Linked** | 网路连线状态 | 是否处于友军 / 中继节点的网路覆盖内 | 链结图示亮 / 暗,断线时无人机控制不可用 |
| **Command Range** | 指挥范围 | 可远程控制 / 指挥的下属单位最大距离 | 小地图上的虚线圆,中心半径数字 |

> **设计补注**:HP 与 Armor 是「死亡导向」,Battery / Heat / Stability 是「节奏导向」,Signal / Linked / Jammed 是「网路战导向」,Sensor / Noise / Command Range 是「资讯战导向」。这四组各自构成独立的 UI 区块。

## 二、武器分类

> **共通栏位**:`name` / `type` / `purpose` / `pros` / `cons` / `view` / `icon` / `look`

### 1. 步枪类 Rifle

#### AR-30 标准突击步枪
- **类型**:Assault Rifle
- **用途**:中距离主力武器,平衡的伤害 / 射速 / 弹匣
- **优点**:全场通用、装填快、稳定度高
- **缺点**:远距离散布大,对装甲目标乏力
- **视角**:FPS / TPP
- **UI icon**:简化步枪剪影 + 弹匣块
- **抽象外观**:长矩形枪管 + 短弹匣盒 + 红色瞄准带,通体哑灰

#### DMR-44 精准步枪
- **类型**:Designated Marksman Rifle
- **用途**:中远距点对点击杀、装甲单位破口
- **优点**:单发高伤、瞄准快速、远距清晰
- **缺点**:射速慢、近战吃亏、Heat 上升快
- **视角**:FPS / TPP / Top-down
- **UI icon**:加长枪管 + 瞄准镜方块
- **抽象外观**:细长黑色枪身 + 顶部凸起瞄准镜方块 + 双脚架

#### SMG-9 短管突击武器
- **类型**:Sub-Machine Gun
- **用途**:近距突袭 / 室内 / 巷战
- **优点**:射速快、Mobility 不打折、近距贴脸高输出
- **缺点**:子弹散布大、远距无效
- **视角**:FPS
- **UI icon**:短粗矩形 + 弹鼓
- **抽象外观**:厚实短身 + 弧形弹鼓 + 双侧红色波浪标志

#### LMG-15 重型自动武器
- **类型**:Light Machine Gun
- **用途**:压制 / 阵地防御 / 蜂群清扫
- **优点**:大弹链、压制效果强、Suppression 输出高
- **缺点**:Mobility 大幅下降、Heat 极快、装填慢
- **视角**:TPP / Top-down
- **UI icon**:粗管 + 弹链图示
- **抽象外观**:加厚枪管 + 长弹链垂下 + 三脚立架

### 2. 发射器类 Launcher

#### RL-7 反载具发射器
- **类型**:Anti-Vehicle Launcher
- **用途**:摧毁 UGV / 砲塔 / 重装机器人
- **优点**:单发巨伤、AOE、可锁定
- **缺点**:载弹少 (3-5)、装填慢、对人形单位过剩
- **视角**:TPP / FPS
- **UI icon**:三角弹头 + 尾焰
- **抽象外观**:管状肩扛 + 尖锥红色弹头 + 后部尾焰孔

#### LM-2 巡飞弹发射器
- **类型**:Loitering Munition Tube
- **用途**:发射后人为引导自杀型无人弹药
- **优点**:可绕过掩体、远距精准、心理压制强
- **缺点**:飞行慢、可被反无人机系统拦截
- **视角**:FPV (引导阶段) / TPP (发射)
- **UI icon**:小型十字飞行器
- **抽象外观**:发射后展开十字翼的小型飞弹,机身红色斜纹

#### DT-3 无人机拋射器
- **类型**:Drone Toss-Cradle
- **用途**:快速将携带的小型无人机抛入空中启动
- **优点**:零延迟部署、不占无人机槽位、可装多种载荷
- **缺点**:射程短 (30 m)、不能用于战斗中重装
- **视角**:TPP
- **UI icon**:箱体 + 上抛箭头
- **抽象外观**:背载箱体 + 翻盖式上半部 + 内置折叠无人机

#### ECM-G 烟雾 / 电磁弹发射器
- **类型**:Smoke / EM Grenade Launcher
- **用途**:快速创造视觉 / 电子掩护,中断敌方锁定
- **优点**:多用途榴弹 (烟雾 / EMP / 闪光 / 热诱饵)
- **缺点**:不造成直接伤害,需配合武器
- **视角**:FPS / TPP
- **UI icon**:小榴弹 + 同心圆扩散
- **抽象外观**:粗管榴弹发射器,弹头六角形,色码标识弹种

### 3. 无人机武器 Drone Weapon

#### FPV-K1 自杀式无人机
- **类型**:Kamikaze Drone
- **用途**:玩家亲自驾驶撞击高价值目标
- **优点**:精准、可绕过掩体、撞击 AOE 大
- **缺点**:一次性、操作期间玩家本体易暴露
- **视角**:FPV
- **UI icon**:四旋翼 + 红色弹头
- **抽象外观**:四旋翼 + 前端红色锥形战斗部 + LOCK 显示

#### RC-S 侦察无人机
- **类型**:Recon Drone
- **用途**:扩大 Sensor 范围、标记敌人、热成像扫描
- **优点**:无武装但隐蔽、电池长、可悬停
- **缺点**:无攻击能力、被反无人机系统针对
- **视角**:UAV (顶视) / Top-down
- **UI icon**:四旋翼 + 摄影镜头方块
- **抽象外观**:扁平四旋翼 + 下挂方形摄像头 + 蓝色信号灯

#### DB-A 投放式无人机 (Bomber)
- **类型**:Drop Bomber Drone
- **用途**:空中投放小型炸弹 / 反装甲弹 / EMP 罐
- **优点**:可重复使用、垂直攻击屋顶 / 工事
- **缺点**:载弹有限、需要瞄准时间
- **视角**:UAV / FPV
- **UI icon**:无人机底部挂载弹头
- **抽象外观**:六旋翼 + 腹部双弹挂架 + 红色挂载箱

#### SW-N 蜂群节点无人机
- **类型**:Swarm Node Drone
- **用途**:释放 4-8 架小型协同无人机蜂群
- **优点**:数量优势、自主分散攻击、难以全数拦截
- **缺点**:单体脆弱、协同被 EMP 一次破坏
- **视角**:RTS / Top-down
- **UI icon**:大圆 + 周围小点
- **抽象外观**:中央母机扁平菱形,周围小型三角附属机

#### RL-3 通讯中继无人机
- **类型**:Relay Drone
- **用途**:扩大 Signal / Linked 网路覆盖范围
- **优点**:大幅延伸控制半径、为团队提供网路掩蔽
- **缺点**:被破坏即失联、不能攻击、容易被反无人机锁定
- **视角**:RTS
- **UI icon**:无人机 + 同心圆波纹
- **抽象外观**:细长机身 + 顶部 V 形天线 + 蓝色脉冲光晕

#### DC-Y 诱饵无人机
- **类型**:Decoy Drone
- **用途**:模拟玩家 / 友军讯号热源,吸引敌方火力
- **优点**:消耗敌方弹药、暴露火力点位置
- **缺点**:近距离立即穿帮、操作时本体暴露
- **视角**:RTS / Top-down
- **UI icon**:无人机 + 问号 / 假身份图示
- **抽象外观**:基础四旋翼但贴着发光红色标志,故意明显

### 4. 机器人武器 Robotic Weapon

#### WLF-Mount 机器狼背载武器
- **类型**:Quadruped Mounted Cannon
- **用途**:M-WLF 背上的轻型自动炮 / 反装甲弹发射筒
- **优点**:跟随移动、低姿态、稳定性高
- **缺点**:射界受机身限制、热量极快累积
- **视角**:TPP (操控狼) / RTS
- **UI icon**:简化狼形 + 顶部砲管
- **抽象外观**:四足机身 + 上方旋转砲座 + 双管或单管

#### UGV-T7 无人车砲塔
- **类型**:Vehicle Turret
- **用途**:无人轮 / 履带车顶部的远程武器站
- **优点**:重装甲底盘、长射程、可载乘员
- **缺点**:目标巨大、对屋顶来袭无能
- **视角**:TPP / Top-down / Turret
- **UI icon**:箱型车 + 圆形砲塔
- **抽象外观**:低矮装甲箱 + 360° 圆形砲塔 + 两侧红色编号

#### ASE-2 自动哨戒砲塔
- **类型**:Autonomous Sentry
- **用途**:玩家部署后自主侦测 / 攻击敌方
- **优点**:守口铁壁、不消耗玩家注意力
- **缺点**:固定不能移动、被破坏后需要重新放置
- **视角**:Top-down / Turret
- **UI icon**:三角架 + 圆形雷达
- **抽象外观**:三脚架 + 中央方块视觉单元 + 单管武器 + 旋转底座

#### RT-W 屋顶武器站
- **类型**:Rooftop Weapon Station
- **用途**:固定建筑顶端的远程支援火力点
- **优点**:制高点、视野极佳、AP 弹可穿掩体
- **缺点**:大型固定目标、容易被 LM-2 拆除
- **视角**:TPP / RTS
- **UI icon**:屋顶 + 砲身
- **抽象外观**:水泥盒底座 + 装甲护盾 + 上方机砲与瞄准球

#### GFP-1 地面遥控火力点
- **类型**:Ground Fire Point
- **用途**:埋伏式可遥控发射器,潜伏地下或遮蔽物
- **优点**:奇袭、低成本、伪装性高
- **缺点**:单次使用、暴露后无法回收
- **视角**:Turret / Top-down
- **UI icon**:地面孔洞 + 弹道箭头
- **抽象外观**:嵌入地面的扁平六角片 + 中央可弹出的发射筒

### 5. 电子战装备 Electronic Warfare

#### JM-X 讯号干扰器
- **类型**:Signal Jammer
- **用途**:压制半径内敌方 Signal / Linked 状态
- **优点**:强迫敌方 EW 失效、阻断敌方无人机操作
- **缺点**:也压制自己,需要友军在干扰区外接力
- **视角**:RTS
- **UI icon**:同心圆 + 中央 X
- **抽象外观**:背包式 + 顶部三根天线呈三角形竖立 + 红色脉动核心

#### GPS-D 位置欺骗器
- **类型**:GPS Spoofer
- **用途**:让敌方导引武器 / 自动单位的座标偏移
- **优点**:误导巡飞弹、制造内伤、隐蔽运作
- **缺点**:作用慢、对手动操作目标无效
- **视角**:RTS
- **UI icon**:卫星 + 偏移箭头
- **抽象外观**:扁平方块装置 + 顶部小型碟形天线缓慢旋转

#### TH-D 热源诱饵
- **类型**:Thermal Decoy
- **用途**:释放高热假目标,扰乱热成像 / 红外锁定
- **优点**:被动消耗、对各种热武器都有效
- **缺点**:可见光照样能看到玩家、燃烧时间短
- **视角**:FPS / TPP
- **UI icon**:火焰 + 散射波纹
- **抽象外观**:小型罐体 + 顶部喷出橙红色火焰柱 + 散热鳍片

#### RD-Pulse 雷达扫描器
- **类型**:Radar Sweeper
- **用途**:主动雷达扫描,大范围标记隐蔽单位
- **优点**:可侦测建筑后的敌人、扩大共享地图
- **缺点**:开启时立即暴露自身位置 (高 Noise)
- **视角**:Top-down / RTS
- **UI icon**:扫描扇形
- **抽象外观**:三脚架 + 中央旋转方形雷达板 + 红色脉冲圆环

#### AD-Net 无人机反制器
- **类型**:Anti-Drone Net
- **用途**:发射缠绕网或 EMP 脉冲,瘫痪敌无人机
- **优点**:针对蜂群效果极好、可救援己方
- **缺点**:射程短、对地面目标无效
- **视角**:FPS / TPP
- **UI icon**:网格 + 无人机被困
- **抽象外观**:类似 RL-7 但发射端为收缩式六角网格筒

#### FT-G 假目标投放器
- **类型**:False Target Generator
- **用途**:在地图上投放虚假信号 / 雷达回波,误导敌方决策
- **优点**:战术欺骗、可设置假阵地、配合伏击
- **缺点**:近距离观察会穿帮、放置时间长
- **视角**:RTS / Top-down
- **UI icon**:幽灵图示 + 雷达回波
- **抽象外观**:小型抛掷弹 + 落地后展开为浮空全息红色方块

## 三、装备配置 Loadout

### 装备槽位 (10 格)

| 槽位 | 英文 | 描述 |
|---|---|---|
| 主武器 | Primary Weapon | 主战斗武器 (步枪 / 发射器) |
| 副武器 | Secondary Weapon | 备用近战 / 短管武器 |
| 无人机槽 | Drone Slot | 携带的可部署无人机 |
| 工具槽 | Utility Slot | 榴弹 / 修理工具 / 维修包 |

| 感测模组 | Sensor Module | 决定 Sensor 范围与解析度 |

| 电池模组 | Battery Module | 决定 Battery 上限 / 充能速度 |
| 指挥模组 | Command Module | 决定 Command Range / 可控单位数 |
| 载荷槽 | Payload Slot | 任务专用物品 (核心数据 / 维修件) |

### 5 种预设配置

#### 1. ASSAULT · 突击型
- **战场定位**:正面推进、近中距交火、清扫据点
- **初始装备**:
  - 主武器:AR-30
  - 副武器:SMG-9
  - 无人机:FPV-K1
  - 工具:ECM-G (烟雾)
  - 装甲:中型复合板
  - 感测:基础雷达
  - 机动:标准伺服
  - 电池:标准
  - 指挥:无
  - 载荷:轻装
- **优势**:平衡、易上手、火力稳定
- **弱点**:对载具与远距压制乏力
- **适合玩家**:第一次玩、喜欢硬刚的玩家

#### 2. RECON · 侦察型
- **战场定位**:外圈机动、标记敌人、短促狙杀
- **初始装备**:
  - 主武器:DMR-44
  - 副武器:SMG-9
  - 无人机:RC-S 侦察机
  - 工具:EM 榴弹 (短期 EMP)
  - 装甲:超轻碳纤板
  - 感测:长程光电 / 热成像
  - 机动:运动模组 (1.4×)
  - 电池:长效
  - 指挥:基础信号节点
  - 载荷:最小
- **优势**:速度快、信息广、远距精准
- **弱点**:被贴脸即败、装甲极弱
- **适合玩家**:喜欢风筝 / 走位 / 标记的玩家

#### 3. DRONE OPERATOR · 无人机操作者
- **战场定位**:后方掩体内驾驭多台无人机
- **初始装备**:
  - 主武器:AR-30
  - 副武器:SMG-9
  - 无人机:SW-N 蜂群节点 (4 架小型)
  - 工具:RL-3 中继无人机
  - 装甲:轻量
  - 感测:网路增益模组
  - 机动:标准
  - 电池:大容量
  - 指挥:多无人机控制 (4 同时)
  - 载荷:无人机加挂
- **优势**:同时打击多点、阵地战王者
- **弱点**:本体脆弱、被 EMP 同时瘫痪、Processing Load 高
- **适合玩家**:喜欢经营 / 微操 / 战术节点的玩家

#### 4. HEAVY UNIT · 重装型
- **战场定位**:阵地推进、坦克 / 砲塔击破
- **初始装备**:
  - 主武器:LMG-15
  - 副武器:RL-7
  - 无人机:DB-A (轰炸)
  - 工具:烟雾弹
  - 装甲:重型陶瓷板
  - 感测:基础
  - 机动:负重伺服 (0.7×)
  - 电池:标准
  - 指挥:无
  - 载荷:满载
- **优势**:火力凶猛、能扛、AOE 强
- **弱点**:慢、目标大、Heat 极快
- **适合玩家**:喜欢扎根固守、缓慢推进的玩家

## 五、能量 / 节奏 / 维修系统

> 这不是人类士兵的「体力」,而是机器人化的资源管理系统。

### 1. Battery 电池
- **消耗来源**:冲刺 (×2)、跳跃、控制无人机、电子战开启、热武器射击
- **回复**:充电站 / Engineer 帮你充电 / 待机时缓慢回复 (1 % / 秒)
- **耗尽后果**:Mobility 降至 0.4×、武器无法射击、无人机失联
- **游戏规则**:满 100,大动作 (启动 EW / 发射 RL-7) 一次扣 8-15

### 2. Heat 热量
- **累积来源**:连续射击、衝刺、电子战、奔跑
- **下降**:站立 / 蹲下不动时快速下降 (3 % / 秒)
- **过热后果**:武器锁定 2-3 秒强制冷却、Stability 下降
- **游戏规则**:武器有自己的热条,角色总热量是辅助。两条独立追踪

### 3. Signal 讯号
- **下降来源**:离开友军节点超过 50 m、进入敌方 Jamming 范围
- **上升**:回到节点 / 部署 RL-3 中继 / 站上讯号塔
- **断线后果**:无人机控制失效、小地图变模糊、指挥范围归零
- **游戏规则**:0-100 等级,< 30 时无人机断线,< 10 时 Linked = false

### 4. Stability 稳定度
- **下降来源**:被爆炸命中、EMP、近距离开火被压制
- **上升**:停顿 1-2 秒不动作、维修
- **下降后果**:准星抖动 / 命中率 -30 %、跳跃失误 / Mobility 下降
- **游戏规则**:0-100,80 以上无影响,以下逐步降级,30 以下严重抖动

### 5. Processing Load 运算负载
- **上升来源**:同时控制无人机数量、开启感测、运行 RTS 视角
- **下降**:关闭无人机控制、放下指挥模组
- **过载后果**:无人机控制延迟、雷达更新降速、冷却时间变长
- **游戏规则**:每个无人机 +20 负载,> 80 % 进入「饱和」状态

### 6. Repair State 维修状态
- **部位**:头 (Sensor)、躯干 (HP / Battery)、臂 (Weapon Heat)、腿 (Mobility)
- **来源**:被命中后该部位累积「损伤等级」(0/1/2/3)
- **影响**:每级该部位对应数值 -15 %
- **修复**:自助慢速修复 (5 秒 / 级) 或 Repair Android / Engineer 快速修复 (1 秒 / 级)

## 六、任务模式

### 1. DOMINATION · 佔点
- **玩法一句话**:占领并守住地图上 3 个控制点 (A / B / C)
- **主要目标**:每个被占领的控制点会持续提供分数
- **常见单位**:Rifle Android / Auto Turret / Smart Barricade
- **适合视角**:TPP / FPS,可切换 RTS 检视控制点状态
- **胜利条件**:分数先达到 1000
- **失败条件**:对方分数先到 1000 或全队失能
- **UI 重点**:顶部 A / B / C 圆形进度条、各控制点距离与持续秒数

### 2. CONVOY ESCORT · 护送无人补给车
- **玩法一句话**:陪着 UGV-Cargo 从起点开到终点,中途敌人会埋伏
- **主要目标**:让运输车存活到达指定地点
- **常见单位**:UGV-Cargo (友) / Sentry Crawler / FPV Drone / Drone Hive
- **适合视角**:TPP (跟随)、UAV (空中支援)
- **胜利条件**:运输车 HP > 0 并抵达终点
- **失败条件**:运输车 HP 归零或所有玩家失能
- **UI 重点**:运输车 HP 条、地图上路线指示、距离终点公尺数

### 3. DRONE HUNT · 猎杀敌方无人机
- **玩法一句话**:在限定时间内击落最多敌方无人机
- **主要目标**:累积无人机击坠数
- **常见单位**:FPV Drone / Recon Drone / Swarm Node / Decoy Drone
- **适合视角**:FPS / TPP (空对空)、UAV (反制)
- **胜利条件**:5 分钟内击落数高于敌方
- **失败条件**:玩家被无人机撞坠 3 次
- **UI 重点**:大屏幕中央击坠计数、剩余时间、敌方蜂群预警

### 4. SIGNAL WAR · 争夺讯号塔
- **玩法一句话**:占领讯号塔扩大网路控制权
- **主要目标**:控制 3 座 Signal Tower 中的多数
- **常见单位**:Signal Android / Relay Drone / Jamming Station / Signal Tower
- **适合视角**:RTS / TPP
- **胜利条件**:控制 2 座以上塔达 90 秒
- **失败条件**:全部塔被夺回或队伍 Signal 全失
- **UI 重点**:每座塔的所属指示灯、Signal 网路覆盖范围着色 (蓝 / 红)

### 5. HIVE RAID · 摧毁无人机蜂巢
- **玩法一句话**:深入敌区破坏 Drone Hive 主结构
- **主要目标**:摧毁 1 个核心 Hive (HP 8000+)
- **常见单位**:Drone Hive / Swarm Node Drone / Heavy Android / Auto Turret
- **适合视角**:TPP / FPV (用 FPV 自杀机攻击核心)
- **胜利条件**:Hive 核心 HP 归零
- **失败条件**:全队失能或被增援无人机海淹没
- **UI 重点**:Hive 大型 HP 条、剩余生产时限、自身蜂群压力指示

### 6. EXTRACTION · 回收核心资料
- **玩法一句话**:潜入资料中心夺取核心,带回回收点
- **主要目标**:取得 Data Core (载荷槽),回到撤离区
- **常见单位**:Sniper Android / Breacher Android / Heavy Android / Auto Turret
- **适合视角**:FPS / TPP
- **胜利条件**:Data Core 抵达撤离区
- **失败条件**:携带者死亡且核心被敌方夺回
- **UI 重点**:核心持有者标记、撤离区距离、隐蔽指示 (Noise / Sensor)

### 7. URBAN SWEEP · 清扫城市区块
- **玩法一句话**:逐街区清除所有敌对单位
- **主要目标**:依序清空 5 个区块的敌人
- **常见单位**:全种类敌人 (随区块变化)
- **适合视角**:TPP / FPS / Top-down
- **胜利条件**:5 个区块全部清空
- **失败条件**:全队失能或时间耗尽 (15 分钟)
- **UI 重点**:区块进度条、当前区块剩余敌人数、增援警告

### 8. RTS COMMANDER · 指挥多单位作战
- **玩法一句话**:从战术地图视角控制整个小队
- **主要目标**:依任务可变 (占点 / 护送 / 摧毁)
- **常见单位**:全种类友军 (玩家可命令)
- **适合视角**:Top-down / RTS (主) + 可切换 FPV / TPP 直接介入
- **胜利条件**:完成主目标且至少 50 % 单位存活
- **失败条件**:主目标失败或全队损失
- **UI 重点**:战术地图 + 单位列表 + 选定单位状态 + 命令面板 (移动 / 攻击 / 部署)

## 附录:风格统一性指引

- **主调**:#1A1A1A 黑、#6A6A6A 灰、#D0D0D0 白、#C8261C 红
- **形体**:所有单位 / 物件由 BoxGeometry 组成,角度只用 0° / 45° / 90°
- **材质**:Lambert 平面阴影,无金属反光,极少使用透明度
- **比例**:每个单位都用「方块的整数倍」(0.2 / 0.4 / 0.8 / 1.6 m) 构成
- **红色用法**:仅限于威胁标记 / 玩家受伤反馈 / 关键互动物 / 敌方识别
- **典型 Icon 风格**:8×8 ~ 16×16 像素方块图,纯白线条 + 红色重点
